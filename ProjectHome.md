# C# World Of Warcraft Armory Library #

This library allows you to query the [World Of Warcraft Armory](http://eu.wowarmory.com) from your projects.


---


## Supported Regions ##
  * USA
  * Oceanic
  * Europe
  * Korea
  * China
  * Taiwan

## Available Features ##
  * Guild
    * Search
    * Roster
  * Character
    * Search
    * Basic Sheet (Except Items)
